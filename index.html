Local-first risk intelligence for insurance ops.

Import policyholder CSVs, impute missing fields, compute LTV, apply JSON rules for risk tiers, build segments, and simulate cross-sell campaigns — all in PHP + SQLite.

Local-first SQLite CSRF Rules engine Segments Campaign simulator

Workflow

Import → clean → score → segment → simulate. All local by default.

Import + impute

Upload CSV, map columns, preview changes, and impute missing values with mean/mode/defaults.

Rules → risk tiers

Edit JSON rules, test on a record, then apply in priority order to set tier + tags.

Segments

Build segments from rule sets + a safe filter expression, estimate size, and export CSV.

Campaign simulator

Model cross-sell assumptions and get gross, cost, and net projections before sending.

Stack

Portable by design. Runs on shared hosting with only built-in PHP extensions.

Runtime
PHP 8+ (PDO + SQLite)
Database
SQLite (WAL + NORMAL)
Security
CSRF + CSP (nonce) + rate limits
Ops
cPanel cron (recompute + segments)
