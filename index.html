Local-first Offline SQLite CSRF Rules Segments/Campaigns

Workflow

Import → clean → score → segment → simulate. All local by default.

Import + impute

Upload CSV, map columns, preview, and impute missing values (mean/mode/defaults) with confidence scoring.

Risk tiers

Create JSON rules, test them against a record, and evaluate in priority order to set tier + tags.

Segments

Combine rules and a safe filter expression, estimate size, then export segment CSV with LTV and tier.

Campaign simulator

Project conversions, gross, cost, and net revenue from cross-sell assumptions before sending.

Stack

Portable single-file architecture with optional cron for scheduled recompute.
